#### Tampermonkey (TM)简介

Tampermonkey 是一款免费的浏览器扩展和最为流行的用户脚本管理器，可以在与规则匹配的网页执行任何js脚本。它适用于 Chrome, Microsoft Edge, Safari, Opera Next, 和 Firefox。

##### Tampermonkey (TM)官网：[www.tampermonkey.net](http://www.tampermonkey.net/)
